`faaah` is a "meme utility" Flutter package designed to add humor to the development and debugging process. It plays a loud, iconic "FAAAAH" sound effect whenever a Flutter error is triggered or when called manually.

## ğŸ—ï¸ Technical Architecture

### 1. Core Logic (`lib/faaah.dart`)
- **Engine**: Uses `audioplayers: ^6.5.1` for high-performance audio playback.
- **Global Listener**: Uses `FlutterError.onError` to intercept all framework-level errors. This is non-destructive, meaning it wraps the original error handler so logs still appear as normal.
- **Package Asset Management**: Implements specialized pathing (`packages/faaah/assets/`) to ensure the sound works even when the package is imported into other projects as a dependency.

### 2. Assets (`assets/faaah.mp3`)
- Contains a high-quality 32kbps MP3 file.
- Registered in the package's `pubspec.yaml` so it is bundled with the library.

### 3. Example Project (`example/`)
- A "Calculator" demo that showcases both use cases:
    - **Global**: `Faaah.enableGlobalErrorSound()` in `main()`.
    - **Manual**: `Faaah.play()` inside a `try-catch` block triggered by division-by-zero or parsing errors.
